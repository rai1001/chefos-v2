# UI North Star (Referencia ChefOS)

## Estilo
- Dark premium, glassmorphism controlado
- Blur sutil en fondos, gradientes fríos (azules/teal), bordes finos
- Tipografía limpia, jerarquía fuerte (H1 grande + subtítulos suaves)
- Densidad alta sin ruido: tablas con líneas sutiles, spacing consistente
- Estados semánticos: verde (ok), ámbar (acción), rojo (crítico)
- Sombras suaves (no duras), elevación en cards

## Layout
- App shell consistente:
  - Sidebar o topbar (pero un solo patrón global)
  - Search global persistente
  - Selector de sucursal (multi-branch)
  - Menú usuario
  - Toggle Kitchen Mode

## Interacciones
- Hover sutil en filas y cards
- Botones principales con glow suave (no exagerado)
- Badges claros, legibles
- Acciones críticas requieren confirmación contextual (item + sucursal)
- Command Palette Ctrl/Cmd+K

## Accesibilidad / Cocina
- Kitchen Mode:
  - Contraste y tamaño táctil aumentados
  - Targets grandes (mín 44px)
  - Texto más claro
